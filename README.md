💎 Jewellery Inventory Management System 💎

A Java Swing based desktop application for managing jewellery inventory using a custom Hash Table implementation.

## Features

- Add Jewellery Items
- Search Items by ID
- Update Item Details
- Delete Items
- Sell Jewellery and Update Stock
- Custom Hash Table Implementation
- Collision Handling using Linked List Chaining
- Luxury Styled Java Swing GUI

## Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)
- Hash Tables
- Linked Lists
- Data Structures & Algorithms

## Project Highlights

- Built without using Java HashMap
- Implemented custom hash table from scratch
- Supports CRUD operations
- GUI based desktop application
- Inventory stock management

## Learning Outcomes

- Data Structures Implementation
- Hashing Techniques
- Collision Resolution
- Java Swing GUI Development
- Object-Oriented Design Principles
